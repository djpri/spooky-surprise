# Story Notes Index

Use this folder as the living story bible for The Corrupted Veil of Alderthorn. Material is organized for quick reference while writing narrative, scenes, or systems.

- `background.md` — Setting overview and the Prologue vignette.
- `characters.md` — Player, major NPCs, and bestiary entries.
- `state-tree.md` — Branching narrative map and endings at a glance.
- `world.md` — Worldbuilding details: Veil, rituals, cosmology, cycles.
- `lenses.md` — Rules and narrative effects of the three rings.
- `locations.md` — Key places with sensory anchors and story hooks.
- `factions.md` — Groups, beliefs, tensions, and conflicts.
- `themes-and-tone.md` — Core themes, tone, motifs, scare design.
- `plot-outline.md` — High‑level act structure and beats.
- `plot-outline.md` — Branch‑based outlines and beats.
 - `endings.md` — Terminal outcomes per branch with tone tags and unlock cues.
- `glossary.md` — Canonical terms, names, and pronunciations.
 - `scenes.md` — Short scene drafts showing core mechanics.
 - `folder-structure.md` — Guide to generating the story folder structure.

Conventions across notes:

- Second‑person, present‑tense narration for in‑story text.
- Canonical place name: Alderthorn; demonym: Alderthornians.
- Lens names: Nèamh (white/gold), Muir (light blue), Talamh (black/violet).
- Creed: “Lux beò, solas mòr, glainne dorcha.”
